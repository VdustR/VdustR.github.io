# Fish Shell

Official site: https://fishshell.com/ (opens new window)

# Install

Mac:

brew install fish
1

# Fisher

Fisher (opens new window) is a package manager for the fish shell.

Install:

curl https://git.io/fisher --create-dirs -sLo ~/.config/fish/functions/fisher.fish
1

If you can't use fisher command, try re-login the shell.

# Spacefish

Spacefish (opens new window) is a minimalistic, powerful and extremely customizable Fish Shell prompt. It combines everything you may need for convenient work, without unnecessary complications, like a real spacefish.

fisher add matchai/spacefish
1

# Fixing Perl Warning: Setting locale failed on Mac OS

You might see this while you execute fisher:

perl: warning: Setting locale failed.
perl: warning: Please check that your locale settings:
        LC_ALL = (unset),
        LANG = "en_TW.UTF-8"
    are supported and installed on your system.
1
2
3
4
5

Edit fish config:

vi ~/.config/fish/config.fish
1

Add locale environment variables:

+ export LC_CTYPE=en_US.UTF-8
+ export LC_ALL=en_US.UTF-8
1
2

Reference: Fixing Perl Warning: Setting locale failed on Mac OS X Mavericks (opens new window)

# Integrate with nvm

Node Version Manager (opens new window) - Simple bash script to manage multiple active node.js versions

To install or update nvm, you can use the install script using cURL:

curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash
1

or Wget:

wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash
1

Then install fish-nvm (opens new window):

fisher add FabioAntunes/fish-nvm edc/bass
1

If node is not loaded at start, create a init script:

vi ~/.config/fish/conf.d/nvm.fish
1

and enter:

nvm use node
1

If you always want to use the latest version, use the followed scripts instead:

nvm install node
# latest npm and yarn
npm i -g npm yarn
1
2
3

then try another session.

# Windows alternative

Cmder (opens new window) is a software package created out of pure frustration over the absence of nice console emulators on Windows. It is based on amazing software, and spiced up with the Monokai color scheme and a custom prompt layout, looking sexy from the start.
